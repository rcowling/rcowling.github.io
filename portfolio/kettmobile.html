Keweenaw Time Traveler: Mobile Storytelling

A web mobile application that allows users to explore the Copper Country through historic maps, aerial imagery, and stories. Users can submit stories right from their mobile device. The application was built using ArcGIS JavaScript SDK and Ionic mobile framework. A basic shell of this application was built and used at the 2025 NEH Community Deep Mapping Institute allowing fellows to easily create their own deep mapping mobile applications!
