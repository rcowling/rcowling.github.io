Building a Deep Mapping Mobile Application

This workshop was designed for the 2025 NEH Community Deep Mapping Institute at Michigan Technological University. In this workshop fellows design their own GIS mobile deep mapping applications using a custom-designed shell in HTML, CSS, and JavaScript. Fellows can easily style and configure the app using a JSON configuration file. Alternative non-code examples are explored using ArcGIS Experience Builder to create a deep map.
