L'Anse, Michigan Sanborn Maps

This interactive map shows what L'Anse, Michigan looked like in 1938 with the use of Sanborn Fire Insurance Maps. The scanned map were downloaded from the Library of Congress and geofererenced in ArcGIS Pro and then clipped and mosaiced. The mosaic was uploaded to Mapbox and served as raster tiles. The application allows for a look into the past by either using the opacity slider or the spy glass. The application was built using the OpenLayers API and Bootstrap Materialize.Click Here to view the interactive map!
