AGSL Nautical Chart Explorer

This map was created for a library fieldwork experience at the American Geographic Society Library (AGSL). The interactive mapping application allows users to discover nautical charts in the AGSL collections. The aim of this application was to make a discovery tool that was user friendly and catered to the general public. The user has the ability to view a high-resoltion image of each nautical chart right in the application. The application was created using the ArcGIS Online JavaScript API 4 and Bootstrap. The project is an early use case of the OpenIndexMaps schema. Click Here to view the interactive map!
